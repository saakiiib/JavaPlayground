# Contact Management System

Welcome to the Contact Management System repository! This project allows you to manage and store contact information using Java. Whether you're new to Java or an experienced developer, this simple application serves as a playground for practicing your coding skills.

## Features

- Add new contacts with names, phone numbers, and email addresses.
- View a list of stored contacts and their basic details.
- Search for a contact by name and view their details.
- Update contact details, such as phone numbers and email addresses.
- Delete unwanted contacts from the system.
- Contacts are persisted between runs using the `contacts.txt` file.

## How to Use

1. Clone the repository to your local machine.
2. Compile and run the `Main.java` file using your preferred Java development environment.
3. Follow the on-screen menu options to interact with the Contact Management System.
